프로세스빌더 툴바
 
프로세스빌더 툴바는 프로세스업무를 편집하기 위한 메뉴들입니다. 프로세스빌더 툴바에 있는 메뉴를 이용하여 작업 한 내용을 프로세스에디터에서 바로 확인이 가능하며, 프로세스에디터에서 편집된 내용 그대로 업무공간에서 작업이 됩니다.
 
 
[그림] 프로세스툴바
 
프로세스툴바 메뉴 소개
 

저장

프로세스 다이어그램을 저장합니다.

뒤로가기

이전 편집한 상태로 이동합니다.

앞으로가기

앞서 편집한 상태로 이동합니다.

회전하기

풀의 정렬을 세로 혹은 가로 형태로 이동합니다.

확대보기

프로세스에디터를 확대하여 보여줍니다.

축소보기

프로세스에디터를 축소하여 보여줍니다.

좌측정렬

2개 이상 태스크가 선택 되었을 경우, 태스크들을 좌측으로 정렬해줍니다.

가로중앙정렬

2개 이상 태스크가 선택 되었을 경우, 태스크들을 가로중앙으로 정렬해줍니다.

우측정렬

2개 태스크가 선택 되었을 경우, 태스크들을 우측으로 정렬해줍니다.

상단정렬

2개 이상 태스크가 선택 되었을 경우, 태스크들을 상단으로 정렬해줍니다.

세로중앙정렬

2개 이상 태스크가 선택 되었을 경우, 태스크들을 세로중앙으로 정렬해줍니다.

하단정렬

2개 이상 태스크가 선택 되었을 경우, 태스크들을 하단으로 정렬해줍니다.

레인중앙정렬

2개 이상 태스크가 선택 되었을 경우, 태스크들을 레인 중앙으로 정렬해줍니다.

선택

태스크들을 클릭하여 선택 합니다.

연결하기

태스크들을 화살표 라인으로 연결 할 수 있습니다. 순차적 절차를 표현합니다.

레인 추가

레인을 추가해 줍니다.

태스트 추가

작업이 추가 됩니다. 작업에는 업무 처리자와 업무화면이 설정 되어야 합니다.

시작이벤트추가

시작이벤트를 추가합니다.

종료이벤트추가

종료이벤트를 추가합니다.

선택분기추가

선택분기를 추가합니다. 선택 분기는 분기되는 경로 중 하나의 경로만 실행됩니다.

병렬분기추가

병렬분기를 추가합니다. 병렬 분기는 분기되는 모든 경로를 병렬로 실행됩니다.

데이터오브젝트추가

데이터 오브젝트를 추가합니다. 데이터 오브젝트는 데이터의 입출력을 표현합니다.

그룹추가

그룹을 추가합니다. 다이어그램의 특정 오브젝트들을 그룹핑 표기해 줍니다.

주석달기

주석을 추가합니다. 설명이 필요한 부분에 주석을 달 수 있습니다
