Installation
 
설치환경
스마트웍스닷넷은 데스크탑, 노트북, 서버 등 어느 하드웨어에 설치가 가능하며 window xp와 vista, Linux에서도 설치가 가능합니다. 설치서버와 사용자의 컴퓨터는 네트웍이 되야하며 CPU는 펜티엄4이상, 메모리는 1G이상을 권장합니다.
 
제품 설치
 
  • 인스톨 전에 프로그램 파일 폴더 아래 PostgreSQL이란 폴더가 있더면 이 폴더를 설치전에 제거하여 주시기 바랍니다.
  • 인스톨 후에 컴퓨터를 재부팅하여 윈도우 설정을 변경하여야 합니다. 재부팅을 안 할 경우 오류가 생길 수 있습니다.
     
  • 스마트웍스닷넷 설치를 시작합니다.
  • 설치파일을 실행하면 좌측과 같은 화면이 나타납니다.
  • 다음 버튼을 클릭합니다.
 
 

 

  
  • 라이센스을 확인합니다.
  • 동의 후 다음 버튼을 클릭합니다.
 
 

 

  
  • 설치폴더를 선택하는 화면입니다. 기본설치 폴더를 변경하시려면 찾아보기 버튼을 눌러 설치할 폴더를 선택합니다.
  

 

  
  • 설치방식에는 두 가지가 존재합니다. 기본설정설치 와 사용자지정설치 입니다.
  • 기본 설정 설치 : 스마트웍스닷넷의 구성요소 및 설정정보를 사용자의 설정 없이 기본값으로 자동 설치합니다. 기본 설정 설치를 선택하고 다음을 누르면 설치가 바로 시작됩니다.
  • 사용자 지정 설치 : 스마트웍스닷넷의 구성요소 및 설정정보를 사용자의 환경에 맞게 끔 사용자가 직접 입력 후 설치하는 방식입니다. (스마트웍스닷넷 포트, 데이터베이스 종류 선택 등) 이 경우 SW에 대한 지식이 필요합니다.
 

 

  
  • 스마트웍스닷넷의 구성요소를 선택하여 설치할 수 있는 화면입니다.
  • smartworks.net : 스마트웍스닷넷의 서버가 설치됩니다.
  • JDK1.5.0.11 : 스마트웍스는 JAVA 기반으로 개발 되었기 때문에 이를 실행하기 위해서는 JDK가 필요, 이를 설치합니다.
  • PostgreSql : 스마트웍스닷넷에서 기본적으로 제공하는 데이터베이스인 postgresSql 을 설치합니다. 기존에 사용하고 있는 데이터 베이스가 있다면 선택을 해제하고 다음 화면에서 사용중인 데이터베이스의 정보를 기입합니
  • SETUP_DB_TABLE : 스마트웍스의 기본 테이블과 기본 데이터를 생성합니다.
  

 

  
  • 스마트웍스닷넷의 웹포트 및 데이터베이스 설정을 하는 화면입니다.
  • HTTP PORT : 스마트웍스닷넷이 기본적으로 사용할 웹 포트를 설정합니다. 이 포트는 이후 스마트웍스닷넷에 접근할 때 사용합니다.
  • DB TYPE : 스마트웍스닷넷이 기본적으로 제공하는 데이터베이스는 POSTGRESSQL입니다.(다른 DB 사용도 가능합니다. 구성요소 설정 변경 필요)
  • DB IP : 사용하고자 하는 데이터 베이스가 원격에 존재한다면 원격지의 IP주소를 입력합니다.
  • DB PORT : 사용하고자 하는 DB 포트를 설정
  • DB NAME : 사용하고자 하는 DB 이름을 설정
  • ID/PASSWORD 설정
 

 

  
  • 스마트웍스닷넷의 설치를 시작합니다.
 

 

  
  • 스마트웍스닷넷의 설치가 완료되었습니다. 바탕화면에 스마트웍스닷넷 실행 버튼이 생성됩니다.
  • '마침' 버튼을 눌러 주세요.
