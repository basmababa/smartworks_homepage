업무 상태 변경 - 편집, 시작/중지
 
업무관리소 기능 메뉴 소개
 
- 설명 : 업무의 요약정보와 기본정보를 확인 할 수 있습니다
 
- 서비스 상태 : 업무의 상태 정보를 표시해 줍니다.
 
  • 시작됨 업무가 활성화 된 상태로 해당 업무는 업무공간에서 실행 중인 업무입니다. 이 업무는 수정, 편집할 수가 없습니다. 수정이나 편집 업무를 위해서는 버튼을 클릭하여 업무를 비활성화 상태인 중지됨으로 변경해야 합니다.
  • 중지됨업무가 비활성화 된 상태로 업무를 수정, 편집할 수 있습니다. 버튼을 클릭하면 활성화 됩니다.
 
- 편집상태
 
  • 서비스 상태가 시작됨이면 편집불가, 중지됨이면 편집가능으로 나타납니다.
  • 편집가능상태에서 버튼을 클릭하면 편집을 위한 스마트웍스 빌더가 실행되며 편집중인 상태로 전환됩니다.
  • 편집을 완료한 후 업무의 서비스 상태를 시작됨으로 변경하기 위하여는 편집 완료 후 버튼을 클릭 후 버튼을 순서대로 클릭하여야 합니다.
 
- 업무 편집화면으로 이동합니다. 편집상태가 중지되어 있을 경우에는 편집된 화면만 볼 수 있고 수정은 할 수 없습니다.
- 업무를 복사합니다.
- 업무를 다른 카테고리로 이동합니다.
