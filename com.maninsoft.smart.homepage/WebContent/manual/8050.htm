데이타 가져오기/내보내기(데이타맵핑)
 
시스템으로 업무를 처리하다 보면 데이타와 관련된 업무가 많습니다. 다른업무참조하여 데이타를 가져오기 또는 내보내기를 하여 다양한 데이타 관련 업무를 처리할 수 있습니다. 해당 셀 또는 항목을 클릭하면 데이타 맵핑 관련 메뉴 아이콘이 나타납니다.
 
데이타 맵핑 메뉴

?가져오기 생성 : 업무(프로세스업무, 정보관리업무)화면에서 데이터를 가져올 때 사용함

내보내기 생성 : 정보관리 업무로 데이터를 내보낼 때 사용함. 오른쪽 연동업무화면 탭에서 내보내기를 할 정보관리업무 선택
삭제 : 가져오기 / 내보내기 설정을 삭제합니다.

 
 
[그림 1] 셀 추가/삭제
 
가져오기 업무 설정
가져오기 버튼 을 클릭하면 가져오기 업무 설정을 위한 팝업창이 뜹니다. 테이타의 필드명을 확인하여 필드값을 가져올 수 있고 수식을 이용할 수도 있습니다.
 
 
[그림 2] 가져오기 업무 설정
 
내보내기 업무 설정
내보내기 버튼 을 클릭하면 내보내기 업무 설정을 위한 팝업창이 뜹니다. 내보낼 정보관리 업무의 테이타 필드명을 확인하여 필드값을 내보낼 수 있습니다.
