폼빌더 항목 속성(항목별)
 
폼빌더의 항목뱔러 적용되는 속성입니다.
 
콤보박스
그림과 같이 옵션에 들어갈 목록을 직접 텍스트를 입력하고 추가합니다. 방향키로 콤보박스 목록의 순서를 변경할 수 있습니다.
 
 
[그림] 콤보박스 목록 추가
 
다른업무 참조
참조할 대상 정보관리업무화면을 선택한 후에 선택 값(키 값)으로 들어 갈 필드를 선택하시면 됩니다. 해당 정보관리업무의 데이터를 이용한 연산도 가능합니다.
 
 
[그림] 다른업무참조 속성 설정
 
통화입력
입력할 통화기호를 선택합니다.
 
 
[그림] 통화입력 속성 설정
 
라디오버튼
라디오버튼 라벨 텍스트를 추가할 수 있습니다. (위에는 설정방법, 아래는 표시내용)
 
 
[그림] 라디오 속성 설정
