업무/업무그룹 관리
 
업무 관리 메뉴 소개
 
업무 추가
새로운 업무그룹, 프로세스업무, 정보관리업무를 추가합니다. 버튼을 누르면 아래 [그림1] 과 같은 팝업 창이 뜨고, 업무이름, 업무설명과 업무종류(업무그룹, 프로세스업무, 정보관리업무)를 선택하고 완료합니다..
 
업무 삭제
업무 또는 업무그룹을 삭제합니다. 삭제할 업무명을 클릭 후 메뉴 버튼을 누룹니다.
 
이름바꾸기
업무 또는 업무의 이름과 설명을 변경할 수 있습니다. 변경할 내용을 클릭 후 메뉴를 누면 아래 [그림2]와 같은 팝업 창이 뜨고 변경 내용을 입력 후 확인 버튼을 클릭하면 완료됩니다.
 
 
[그림 1] 새 업무 만들기
 
 
[그림 2] 업무 수정하기
 
업무 복사
복사 할 대상 업무를 선택한 후 버튼을 누르면 업무가 같은 카테고리 내에 ‘사본_기존이름’ 의 이름을 가진 업무로 복사됩니다.
 
다른 카테고리로 업무 복사
선택 된 카테고리에 [그림 3] 과 같은 팝업 창에서 선택된 업무를 선택한 카테고리로 복사합니다
