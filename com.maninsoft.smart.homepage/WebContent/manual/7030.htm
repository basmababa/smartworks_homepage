폼빌더 항목 속성(공통)
 
폼빌더의 항목은 속성에 모두 적용되는 공통되는 속성입니다.
 
 
[그림] 폼 빌더 항목 공통 속성
 
폼 빌더 항목 공통 속성
 
  • 항목명 : 항목 왼쪽 라벨로 표시됩니다. 텍스트를 직접 입력합니다.
  • 정렬 : 좌/우/가운데 정렬
  • 항목 숨김 : 항목을 실행모드에서 숨겨줍니다.
  • 필수 항목 : 업무 실행시 반드시 입력하여야 할 필수 항목을 설정할 수 있습니다. 업
  • 읽기전용 : 실행 시 입력은 안되고, 읽기만 가능한 항목입니다.
  • 함수 : 현재 사용자, 현재 날짜를 자동으로 입력해 줍니다.
  • 표시방식 : 항목을 다른 항목으로 변경합니다. 항목의 공통속성 이외에 추가 설정을 위한 버튼입니다.
